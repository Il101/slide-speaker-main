### ⭐⭐ Вариант 2: OpenRouter с кредитами

**Стоимость:** $0.30-0.60 per 30 slides (в зависимости от модели)  
**Сложность:** Низкая (2 минуты + пополнение)  
**Качество:** Высокое

#### Что делать:

```bash

#    self.vision_model = "google/gemini-flash-1.5"  # $0.01 per slide
```

**Преимущества:**
- ✅ Простая интеграция (1 строка кода)
- ✅ Экономия 80-90%
- ✅ Разные модели на выбор
- ✅ Уже настроено

**Недостатки:**
- ⚠️ Требует пополнения счёта
- ⚠️ Платная услуга

### ⭐ Вариант 3: Google AI Studio

**Стоимость:** $0.30 per 30 slides или FREE (с limits)  
**Сложность:** Низкая (5 минут)  
**Качество:** Высокое

#### Что делать:

```bash
# 1. Получить API key:
#    https://aistudio.google.com/app/apikey

# 2. Добавить в backend/.env:
GOOGLE_API_KEY=your-key-here

# 3. Изменить semantic_analyzer.py на google.generativeai
```

**Преимущества:**
- ✅ Простая регистрация
- ✅ FREE tier с generous limits
- ✅ Официальный Google API
- ✅ Легко интегрировать

**Недостатки:**
- ⚠️ FREE tier имеет rate limits
- ⚠️ Требует новый API key
