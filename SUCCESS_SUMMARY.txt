❓ Вопрос: Зачем Whisper если Google TTS может timing?
✅ Ответ: Whisper используется ТОЛЬКО с Silero TTS, НЕ с Google!

Логика:
├─ TTS_PROVIDER=google (Production)
│  ├─ Native word_timings через SSML <mark> tags
│  ├─ Whisper НЕ загружается
│  └─ Экономия памяти: ~500MB
│
└─ TTS_PROVIDER=silero (Free fallback)
   ├─ НЕ поддерживает word timing
   ├─ Whisper загружается автоматически
   └─ Извлекает timing из аудио

Текущий Docker: TTS_PROVIDER=google → Whisper НЕ используется ✅
