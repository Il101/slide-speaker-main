# Новая логика генерации лекций с анти-чтением

## Обзор изменений

Реализована кардинально новая логика генерации лекций, которая заменяет простое чтение слайдов на объяснение концептов с проверкой анти-чтения.

## Ключевые изменения

### 1. Извлечение концептов вместо полного текста

**Было:** Модель получала весь текст слайда
**Стало:** Модель получает структурированные концепты:
- Заголовок слайда
- 2-5 ключевых тезисов
- Инсайт по диаграмме/таблице
- Термины для определения

### 2. Анти-чтение с автоматической регенерацией

**Новая логика:**
- Проверка перекрытия с текстом слайда (Jaccard similarity)
- Порог перекрытия: 0.35 (настраивается)
- Автоматическая регенерация при превышении порога
- Усиленные инструкции при повторной генерации

### 3. Структурированный talk_track

**Новая структура речи:**
```json
{
  "talk_track": [
    {"kind": "hook", "text": "Зачем слушателю это знать"},
    {"kind": "core", "text": "Объяснение простыми словами"},
    {"kind": "example", "text": "Пример или аналогия"},
    {"kind": "contrast", "text": "Типичная ошибка/грань"},
    {"kind": "takeaway", "text": "Что запомнить"},
    {"kind": "question", "text": "Риторический вопрос"}
  ]
}
```

### 4. Visual cues для подсветки

**Новая система подсветки:**
- Привязка к элементам слайда
- Синхронизация с talk_track
- Автоматическое определение таймингов

### 5. План лекции для связности

**Глобальный промпт:**
- Генерация общего плана лекции
- Цели для каждого слайда
- Правила повествования
- Обеспечение связности между слайдами

## Архитектура

### Новые модули

1. **`concept_extractor.py`** - Извлечение концептов из слайдов
2. **`ai_generator.py`** - Обновленный генератор с анти-чтением
3. **`v2_lecture.py`** - Новые API endpoints
4. **Обновленные схемы** - Поддержка новой структуры данных

### Обновленные компоненты

1. **`document_parser.py`** - Извлечение концептов
2. **`Player.tsx`** - Поддержка visual_cues и talk_track
3. **`api.ts`** - Новые типы и методы API
4. **`schemas.py`** - Расширенные схемы данных

## Пример использования

### Старый подход (чтение слайда):
```
"Эпидерма, мезофилл, устьица..."
```

### Новый подход (объяснение концептов):
```
"У листа две основные ткани: верхняя защитная плёнка и слой клеток, где идёт фотосинтез. На схеме справа — столбчатые клетки ловят свет, губчатые — пропускают газ. Запомните: устьица снизу — так растение экономит воду. Вопрос: почему у пустынных видов устьица открываются ночью?"
```

## API Endpoints

### Новые V2 endpoints:

- `POST /api/v2/lecture-outline` - Генерация плана лекции
- `POST /api/v2/speaker-notes` - Генерация speaker notes с анти-чтением
- `POST /api/v2/regenerate-speaker-notes` - Регенерация с усиленным анти-чтением
- `GET /api/v2/manifest/{lesson_id}` - Получение манифеста с новой структурой

## Настройки качества

### Тумблеры качества:

1. **`anti_reading_threshold`** = 0.35 - Порог перекрытия с текстом
2. **`min_inference_ratio`** = 0.5 - Минимум новой информации
3. **`style_presets`** - Стили презентации:
   - `explanatory` - Объяснительный
   - `storytelling` - Повествовательный
   - `socratic` - Сократический
   - `exam_prep` - Подготовка к экзамену
