# 🔐 Инструкция по авторизации

## Демо-аккаунты для тестирования

### Администратор
- **Email:** `admin@example.com`
- **Пароль:** `admin123`
- **Роль:** Администратор (полный доступ)

### Пользователь
- **Email:** `user@example.com`
- **Пароль:** `user123`
- **Роль:** Пользователь (стандартный доступ)

## Как войти в систему

1. Откройте приложение по адресу: http://localhost:3000
2. Нажмите кнопку **"Войти в систему"** на главной странице
3. Или перейдите напрямую на: http://localhost:3000/login
4. Введите данные одного из демо-аккаунтов
5. Нажмите **"Войти"**

## Быстрый вход

На странице входа есть кнопки быстрого заполнения:
- **"Админ"** - автоматически заполняет поля данными администратора
- **"Пользователь"** - автоматически заполняет поля данными пользователя

## Возможности после входа

### Для всех авторизованных пользователей:
- ✅ Создание новых лекций
- ✅ Загрузка презентаций (PPTX, PDF)
- ✅ Просмотр и редактирование лекций
- ✅ Экспорт в MP4

### Для администраторов дополнительно:
- 🛡️ Доступ к административным функциям
- 📊 Расширенная аналитика
- ⚙️ Управление системой

## Выход из системы

Нажмите кнопку **"Выйти"** в правом верхнем углу навигации.

## Безопасность

- 🔒 Все пароли хешируются с использованием PBKDF2-SHA256
- 🛡️ JWT токены для аутентификации
- 🍪 CSRF защита для всех POST запросов
- ⏰ Токены действительны 30 минут

**Примечание:** Это демо-аккаунты для тестирования. В production среде используйте собственные учетные данные.
