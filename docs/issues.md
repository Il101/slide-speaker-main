## P0 - Критические проблемы (блокируют e2e)

**Файл:** `backend/app/services/sprint1/document_parser.py`

**Файл:** `backend/app/services/sprint2/ai_generator.py`  
**Проблема:** Нет автоматической смены слайдов во время воспроизведения  
**Воспроизведение:** Воспроизвести урок → нет автопереключения между слайдами

### P0-3: Некорректное масштабирование bbox
**Файл:** `src/components/Player.tsx`  
**Проблема:** При изменении размера окна элементы "уезжают" с своих позиций  
**Воспроизведение:** Изменить размер окна браузера → подсветки не совпадают с текстом  
**Исправление:** Исправить calculateScale функцию в Player.tsx
