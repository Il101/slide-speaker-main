# 🎵 SSML ИНТЕГРАЦИЯ ДЛЯ УЛУЧШЕНИЯ РУССКИХ АКЦЕНТОВ

## 🎯 Проблема
Пользователь сообщил о проблемах с русскими акцентами:
- ❌ Неправильные акценты в предложениях
- ❌ Некорректная интонация
- ❌ Неправильные ударения в словах

## 💡 Решение: SSML (Speech Synthesis Markup Language)

### Что такое SSML?
SSML - это стандарт разметки для синтеза речи, который позволяет:
- **Точный контроль** над произношением
- **Настройка скорости** и тона речи
- **Добавление пауз** между предложениями
- **Выделение важных терминов**
- **Улучшение акцентов** и интонаций

## 🔧 Реализация

### 1. LLM Worker с SSML поддержкой
**Файл**: `backend/workers/llm_openrouter_ssml.py`

**Функции**:
- `generate_lecture_text_with_ssml()` - генерирует текст с SSML-разметкой
- `_create_ssml_lecture_text_prompt()` - создает промпт для SSML генерации
- `_clean_ssml_lecture_text()` - очищает и валидирует SSML

**SSML теги для русского языка**:
```xml
<prosody rate="0.95" pitch="-1.0"> - основной текст
<emphasis level="strong"> - важные термины
<break time="0.5s"/> - паузы между предложениями
<prosody rate="0.9"> - медленное произношение сложных слов
<say-as interpret-as="characters"> - аббревиатуры
```

### 2. TTS Worker с SSML поддержкой
**Файл**: `backend/workers/tts_google_ssml.py`

**Функции**:
- `synthesize_speech_with_ssml()` - синтез с SSML
- `synthesize_slide_text_google_ssml()` - синтез слайдов с SSML
- `_extract_text_from_ssml()` - извлечение текста из SSML

### 3. Интеграция в Provider Factory
**Файл**: `backend/app/services/provider_factory.py`

**Новые функции**:
- `generate_lecture_text_with_ssml()` - генерация SSML текста
- `synthesize_slide_text_google_ssml()` - синтез SSML аудио

```xml
</speak>
```

## 🎵 Преимущества SSML

### Для русского языка:
- ✅ **Правильные акценты** - LLM знает, где ставить ударения
- ✅ **Корректная интонация** - естественные интонации предложений
- ✅ **Паузы между предложениями** - более естественная речь
- ✅ **Выделение терминов** - важные слова произносятся с акцентом
- ✅ **Контроль скорости** - медленнее для сложных слов

### Технические преимущества:
- ✅ **Точные тайминги** - каждое предложение имеет временные метки
- ✅ **Гибкая настройка** - можно настроить любой аспект речи
- ✅ **Совместимость** - работает с Google Cloud TTS
- ✅ **Масштабируемость** - легко добавить новые теги
