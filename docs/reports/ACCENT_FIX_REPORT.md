# 🎵 РЕШЕНИЕ ПРОБЛЕМЫ С АКЦЕНТАМИ В РУССКОМ ГОЛОСЕ

## 🔍 Проблема
Chirp v3 HD голос **ru-RU-Chirp3-HD-Charon** имел проблемы с:
- ❌ Неправильными акцентами в предложениях
- ❌ Некорректной интонацией
- ❌ Неправильными ударениями в словах

## 🧪 Анализ проблемы

### Ограничения Chirp v3 HD голосов:
- **НЕ поддерживают параметр `pitch`** - API возвращает ошибку 400
- **Ограниченные настройки** для улучшения акцентов
- **Фиксированные интонации** без возможности настройки

### Преимущества WaveNet голосов:
- ✅ **Полная поддержка всех параметров** (`speaking_rate`, `pitch`)
- ✅ **Гибкая настройка** для улучшения акцентов
- ✅ **Проверенное качество** для русского языка

## 🎯 Решение

### Выбранный голос:
```env
GOOGLE_TTS_VOICE=ru-RU-Wavenet-A
GOOGLE_TTS_SPEAKING_RATE=0.95
GOOGLE_TTS_PITCH=-1.0
```

### Обоснование выбора:

#### 🎵 **ru-RU-Wavenet-A** (женский голос):
- **Лучше справляется с интонациями** русского языка
- **Более естественные акценты** в предложениях
- **Правильные ударения** в словах
- **Поддержка всех параметров настройки**

#### ⚡ **speaking_rate: 0.95** (медленнее):
- **Улучшает акценты** - больше времени на правильную интонацию
- **Более четкое произношение** сложных слов
- **Естественный темп** для лекций

#### 🎼 **pitch: -1.0** (ниже):
- **Более естественное звучание** - не слишком высокий тон
- **Профессиональный звук** для образовательного контента
- **Лучшая разборчивость** речи
