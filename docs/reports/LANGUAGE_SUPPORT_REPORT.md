## 🚀 КАК ИСПОЛЬЗОВАТЬ

### 1. Настройка языка
```bash
# В .env файле
LLM_LANGUAGE=ru  # Русский
LLM_LANGUAGE=en  # Английский
LLM_LANGUAGE=es  # Испанский
LLM_LANGUAGE=fr  # Французский
LLM_LANGUAGE=de  # Немецкий
```

### 2. Перезапуск backend
```bash
# Backend автоматически загрузит новую настройку языка
cd backend && python3 -m uvicorn app.main:app --host 127.0.0.1 --port 8001 --reload
```

### 3. Загрузка презентации
```bash
# LLM будет генерировать контент на выбранном языке
curl -X POST -F "file=@presentation.pptx" http://localhost:8001/upload
```
