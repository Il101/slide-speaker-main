### 🚀 **КАК ИСПОЛЬЗОВАТЬ**

**1. Активировать Vision API:**
```bash
# Уже активирован в Google Cloud Console
```

**2. Запустить с Vision API:**
```bash
cd backend
GOOGLE_APPLICATION_CREDENTIALS=path/to/credentials.json \
OCR_PROVIDER=vision \
python3 -m uvicorn app.main:app --host 127.0.0.1 --port 8001 --reload
```

**3. Загрузить презентацию:**
```bash
curl -X POST "http://localhost:8001/upload" \
  -H "Content-Type: multipart/form-data" \
  -F "file=@your_presentation.pdf"
```
