### Пример сгенерированного SSML:
```xml
<speak>
<prosody rate="0.95" pitch="-1.0">Добрый день, уважаемые студенты!</prosody> 
<break time="0.5s"/>
<prosody rate="0.95" pitch="-1.0">Сегодня мы продолжаем изучение 
<emphasis level="strong">анатомии растений</emphasis>.</prosody> 
<break time="0.5s"/>
<prosody rate="0.9">Обратите внимание</prosody> на важные детали в таблице.
</speak>
```

## 🎵 SSML теги для русского языка

| Тег | Назначение | Пример |
|-----|------------|--------|
| `<prosody rate="0.95" pitch="-1.0">` | Основной текст | Медленнее и ниже |
| `<emphasis level="strong">` | Важные термины | Выделение ударением |
| `<break time="0.5s"/>` | Паузы | Между предложениями |
| `<prosody rate="0.9">` | Сложные слова | Еще медленнее |
| `<say-as interpret-as="characters">` | Аббревиатуры | По буквам |

## 📊 Сравнение качества

| Параметр | До SSML | После SSML |
|----------|---------|------------|
| **Акценты** | ❌ Плохо | ✅ Отлично |

| **Паузы** | ❌ Нет | ✅ Есть |

## 🔧 Техническая реализация

### Автоматическое определение SSML:
```python
if generate_lecture_text_with_ssml and lecture_text.startswith('<speak>'):
    # Используем SSML TTS для лучшего произношения
    audio_path, tts_words = synthesize_slide_text_google_ssml([lecture_text])
    logger.info(f"✅ Used SSML TTS for slide {slide_id}")
else:
    # Fallback к обычному TTS
    audio_path, tts_words = synthesize_slide_text_google([lecture_text])
    logger.info(f"✅ Used regular TTS for slide {slide_id}")
```

### LLM промпт для SSML:
```
Ты - опытный преподаватель русского языка, который создает лекционный текст с SSML-разметкой для правильного произношения.

Используй следующие SSML-теги для улучшения произношения:
- <prosody rate="0.95" pitch="-1.0"> - для основного текста
- <emphasis level="strong"> - для важных терминов
- <break time="0.5s"/> - для пауз между предложениями
- <prosody rate="0.9"> - для медленного произношения сложных слов
```

## 💡 Преимущества SSML

1. **Точный контроль** - LLM может "говорить" голосу, как произносить
2. **Естественность** - более естественная русская речь
3. **Гибкость** - легко настроить любой аспект произношения
4. **Совместимость** - работает с Google Cloud TTS
5. **Масштабируемость** - можно добавить новые теги
