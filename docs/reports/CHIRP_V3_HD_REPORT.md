## 🔄 Как изменить голос

Чтобы попробовать другой Chirp голос, измените в `backend_env_enhanced_hybrid.env`:

```env
# Для женского голоса:
GOOGLE_TTS_VOICE=ru-RU-Chirp3-HD-Leda

# Для другого мужского:
GOOGLE_TTS_VOICE=ru-RU-Chirp3-HD-Puck
```
