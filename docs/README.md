# Документация Slide Speaker

## Структура

### 📋 reports/
Отчеты о тестировании, интеграции и исправлениях:
- Отчеты по функционалу (pipeline, hybrid, vision, TTS)
- Отчеты по исправлениям (bugs, fixes)
- Отчеты по интеграциям (Google Cloud, OpenRouter, SSML)
- Отчеты по тестированию (frontend, backend, CI/CD)

### 📖 guides/
Руководства пользователя и разработчика:
- `DEPLOYMENT_GUIDE.md` - Развертывание приложения
- `DOCKER_README.md` - Работа с Docker
- `AUTH_INSTRUCTIONS.md` - Настройка аутентификации
- `FRONTEND_TESTING_GUIDE.md` - Тестирование frontend
- `SPRINT*.md` - Документация по спринтам

### 🔧 Прочие файлы

- `issues.md` - Список задач
