# ✅ Voice & Volume Fix - Final Solution

## Проблемы

### Проблема 1: Странные изменения голоса
**Симптом:** "голос TTS меняется странно" в определённой фразе про жилкование

**Причина:** Слишком агрессивные настройки prosody для типа сегмента 'emphasis':
- Rate: 88% (очень медленно)
- Pitch: +2st (очень высоко)
- Emphasis: strong (слишком сильно)

**Решение:** Убрали все изменения rate и pitch из emphasis, оставили только moderate emphasis

### Проблема 2: Меняющаяся громкость
**Симптом:** "громкость почему-то тоже меняется"

**Причина:** Изменения `pitch` (высоты тона) в prosody воспринимались как изменения громкости:
- hook: pitch="-1st" → воспринимается как тише
- example: pitch="+1st" → воспринимается как громче
- summary: pitch="-1st" → воспринимается как тише

Хотя технически volume не менялся, **изменения pitch создавали психоакустический эффект разной громкости**.

## Почему pitch влияет на восприятие громкости?

### Психоакустика

**Fletcher-Munson Curves (Equal Loudness Contours):**
- Человеческое ухо воспринимает разные частоты с разной чувствительностью
- Более высокие частоты (higher pitch) воспринимаются как более громкие
- Более низкие частоты (lower pitch) воспринимаются как более тихие

**Пример:**
```
Одинаковая физическая амплитуда (volume):
- 100 Hz (низкий pitch): воспринимается как 50 дБ
- 1000 Hz (средний pitch): воспринимается как 70 дБ ✅
- 4000 Hz (высокий pitch): воспринимается как 80 дБ

Результат: pitch="+1st" звучит ГРОМЧЕ даже при одинаковом volume!
```

### В нашем случае:

**До fix:**
```
hook: pitch="-1st"     → воспринимается тише на ~10-15%
example: pitch="+1st"  → воспринимается громче на ~10-15%
summary: pitch="-1st"  → воспринимается тише на ~10-15%

Разница в восприятии: до 30% между example и hook/summary ❌
```

**После fix:**
```
Все сегменты: pitch = normal (no change)
Разница в восприятии: 0% ✅
```

## Параметры и рекомендации

### Что использовать для интонации:

✅ **РЕКОМЕНДУЕТСЯ:**
1. **rate** (скорость речи):
   - 90-95%: Медленнее для важных моментов
   - 100%: Нормальная скорость
   - 103-105%: Быстрее для примеров, перечислений

2. **emphasis** (интонационное выделение):
   - moderate: Умеренное выделение
   - strong: Только для критически важных моментов (редко)

3. **break** (паузы):
   - 200-300ms: Короткие паузы между предложениями
   - 400-500ms: Средние паузы между абзацами
   - 600-1000ms: Длинные паузы между секциями

❌ **НЕ РЕКОМЕНДУЕТСЯ:**
1. **pitch**: Изменения высоты тона → воспринимается как изменение громкости
2. **volume**: Технически меняет громкость, но не нужен если pitch контролируется

### Допустимые диапазоны:

**Rate:**
- Минимум: 85% (очень медленно, для критически важной информации)
- Нормально: 90-100% (обычная речь)
- Максимум: 110% (быстро, для примеров/перечислений)
- ⚠️ Не использовать <85% или >110% - звучит неестественно

**Emphasis:**
- none: Без выделения (default)
- moderate: Умеренное выделение (основной вариант)
- strong: Сильное выделение (очень редко)
- reduced: Менее выразительно (не используется)

**Break:**
- Минимум: 100ms (короткая пауза)
- Обычно: 200-500ms (между предложениями)
- Максимум: 1000ms (между секциями)
