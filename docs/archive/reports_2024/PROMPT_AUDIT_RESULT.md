## ✅ Код обработки (не промпты):

### `backend/app/services/bullet_point_sync.py`
- **Функция:** Удаляет [visual:], [lang:] markers при нормализации
- **Строки 162-163:**
  ```python
  text = re.sub(r'\[visual:[a-z]{2}\].*?\[/visual\]', '', text)
  text = re.sub(r'\[lang:[a-z]{2}\](.*?)\[/lang\]', r'\1', text)
  ```
- **Это НЕ промпт!** Это обработка старого формата для обратной совместимости

### `backend/app/services/ssml_generator.py`
- **Функция:** Преобразует [lang:XX], [visual:XX] в SSML теги
- **Метод:** `_process_foreign_terms()`

- **Функция:** Ищет упоминания элементов в тексте для синхронизации
- **Много упоминаний:** [lang:XX], [visual:XX], foreign terms
