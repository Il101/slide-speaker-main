## 💻 Использование

### Отследить событие (Frontend)
```typescript
import { trackEvent } from '@/lib/analytics';

// При загрузке презентации
trackEvent.presentationUploaded({
  fileSize: file.size,
  fileName: file.name
});

// При ошибке
trackEvent.error({
  errorType: error.name,
  errorMessage: error.message,
  location: 'upload'
});
```

```python
    lesson_id=lesson_id
)
```

## 🐛 Решение Проблем

### События не отслеживаются
1. Откройте консоль браузера
2. Проверьте вкладку Network
3. Убедитесь что `/api/analytics/track` возвращает 200
4. Проверьте базу: `SELECT * FROM analytics_events;`

### Дашборд пустой
1. Подождите накопления данных (походите по сайту)
2. Проверьте базу: `SELECT COUNT(*) FROM analytics_events;`
3. Убедитесь что вы админ: `SELECT role FROM users WHERE email = 'ваш@email';`

### Миграция не запускается
1. Проверьте подключение к БД: `psql $DATABASE_URL`
2. Проверьте текущую версию: `alembic current`
3. Проверьте логи: `alembic upgrade head --sql`
