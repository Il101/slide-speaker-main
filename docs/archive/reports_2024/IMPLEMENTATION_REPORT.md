## Ключевые особенности реализации

### 1. Концептуальный подход
```python
# Вместо полного текста слайда
slide_text = "Эпидерма, мезофилл, устьица..."

# Извлекаем структурированные концепты
concepts = SlideConcepts(
    title="Histologie des Blattes",
    key_theses=["Épiderme - couche protectrice", "Mésophylle - tissu photosynthétique"],
    visual_insight="Diagramme montre la structure",
    terms_to_define=["Épiderme", "Mésophylle", "Stomates"]
)
```

### 2. Анти-чтение с автоматической проверкой
```python
# Проверяем перекрытие с текстом слайда
should_regenerate, overlap = check_anti_reading(generated_text, slide_text)

if should_regenerate:
    # Регенерируем с усиленными инструкциями
    prompt += AntiReadingDetector().get_regeneration_prompt_addition()
```

### 3. Структурированный talk_track
```json
{
  "talk_track": [
    {"kind": "hook", "text": "У листа две основные ткани..."},
    {"kind": "core", "text": "На схеме справа — столбчатые клетки..."},
    {"kind": "example", "text": "Представьте лист как многоэтажный дом..."},
    {"kind": "contrast", "text": "Не путайте с корнем..."},
    {"kind": "takeaway", "text": "Запомните: устьица снизу..."},
    {"kind": "question", "text": "Вопрос: почему у пустынных видов..."}
  ]
}
```

### 4. Visual cues для синхронизации
```json
{
  "visual_cues": [
    {"at": "hook", "targetId": "title_element"},
    {"at": "core", "targetId": "bullet_1"},
    {"at": "example", "targetId": "diagram_element"}
  ]
}
```

## API Endpoints

### Новые V2 endpoints:
- `POST /api/v2/lecture-outline` - Генерация плана лекции
- `POST /api/v2/speaker-notes` - Генерация с анти-чтением
- `POST /api/v2/regenerate-speaker-notes` - Регенерация
- `GET /api/v2/manifest/{lesson_id}` - Манифест с новой структурой
