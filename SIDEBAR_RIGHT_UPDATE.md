# Изменения интерфейса: Sidebar справа + прокрутка видео

## Что было изменено

### 1. Перемещение sidebar вправо (src/pages/Index.tsx)

Sidebar с библиотекой видео теперь расположен справа от основного контента на всех страницах:

- **Landing page**: Sidebar справа от hero section и features
- **Upload page**: Sidebar справа от формы загрузки
- **Player page**: Sidebar справа от плеера

#### Изменения структуры:
- Добавлен flex контейнер для размещения контента и sidebar
- Основной контент получил класс `flex-1` (занимает доступное пространство)
- Sidebar имеет фиксированную ширину `w-80` (320px)
- Border изменен с `border-r` (справа) на `border-l` (слева)

### 2. Исправлена прокрутка видео (src/components/MyVideosSidebar.tsx)

Список видео теперь прокручивается внутри своего контейнера, а не растягивается на всю страницу:

#### Изменения:
- Добавлен `flex-shrink-0` для заголовка (фиксированный размер)
- Добавлен `min-h-0` для контейнера списка (позволяет уменьшаться)
- Добавлен `bg-background` для лучшей видимости на темных темах
- Список видео использует `overflow-y-auto` внутри `flex-1` контейнера

### 3. Добавлены новые функции (src/pages/Index.tsx)

Добавлены обработчики для работы с sidebar:

```typescript
const handleVideoSelect = (videoLessonId: string) => {
  setLessonId(videoLessonId);
  setAppState('player');
  saveAppState('player', videoLessonId);
};

const handleVideoDownload = (videoUrl: string, title: string) => {
  const link = document.createElement('a');
  link.href = videoUrl;
  link.download = `${title}.mp4`;
  document.body.appendChild(link);
  link.click();
  document.body.removeChild(link);
  toast.success('Загрузка видео начата');
};
```

### 4. Добавлены импорты

Добавлены необходимые компоненты:
- `MyVideosSidebar` - компонент библиотеки видео
- `Navigation` - навигационная панель
- `useAuth` - хук авторизации
- `apiClient` - клиент API

## Технические детали

### CSS классы для sidebar:
- `w-80` - ширина 320px
- `border-l` - граница слева
- `border-border/50` - полупрозрачная граница
- `h-screen` - высота на весь экран
- `flex flex-col` - вертикальная flexbox раскладка

### CSS классы для списка видео:
- `flex-1` - занимает доступное пространство
- `overflow-y-auto` - вертикальная прокрутка
- `min-h-0` - позволяет уменьшаться (важно для flex)
- `p-2` - внутренние отступы
