# 🔑 Как получить Google AI Studio API Key

**Время:** 3 минуты  
**Стоимость:** FREE

---

## 📋 Пошаговая инструкция:

### Шаг 1: Открыть Google AI Studio

```bash
# Откройте в браузере:
https://aistudio.google.com/app/apikey
```

Или просто Google: "google ai studio api key"

---

### Шаг 2: Войти в аккаунт Google

- Используйте любой Google аккаунт
- Желательно тот же, что связан с GCP проектом
- Но можно и другой

---

### Шаг 3: Создать API Key

1. Нажмите кнопку **"Create API Key"** (или "Get API Key")

2. Выберите:
   - **Option 1:** "Create API key in new project" (проще)
   - **Option 2:** "Create API key in existing project" (ваш GCP проект)

3. Нажмите **"Create API key"**

---

### Шаг 4: Скопировать ключ

**ВАЖНО:** Ключ должен выглядеть так:

```
AIzaSyA1B2C3D4E5F6G7H8I9J0K1L2M3N4O5P6Q
```

**Формат:**
- Начинается с `AIzaSy`
- Длина: ~39 символов
- Только английские буквы, цифры, дефисы

**НЕ должно быть:**
- Точек в середине
- Пробелов
- Кавычек
- Других символов

---

### Шаг 5: Проверить ключ

Скопируйте ключ и дайте мне, я проверю!

---

## ⚠️ Распространённые ошибки:

### Ошибка 1: Скопировали с точками
```
❌ НЕПРАВИЛЬНО: AIzaSy...xyz (с точками)
✅ ПРАВИЛЬНО:   AIzaSyABCDEFG123456789 (полный ключ)
```

### Ошибка 2: Скопировали только часть
```
❌ НЕПРАВИЛЬНО: AQ.Ab8RN6IOVCDekQLLgOTp8... (неполный)
✅ ПРАВИЛЬНО:   AIzaSyABCDEFG123456789 (полный, начинается с AIzaSy)
```

### Ошибка 3: Добавили лишние символы
```
❌ НЕПРАВИЛЬНО: "AIzaSy..." (с кавычками)
✅ ПРАВИЛЬНО:   AIzaSyABCDEFG123456789 (без кавычек)
```
